# temp_script
this is a small bash script for display temp in RPi every second

To use this script,

$ git clone https://github.com/k3nw4y/temp_script.git .    

$ cd temp_script/ </br>

$ chmod +x temp.sh </br>

$ ./temp.sh </br>
